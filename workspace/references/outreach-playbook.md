# Outreach Playbook（触达话术手册）

打招呼消息的生成策略与质量标准。

---

## 核心原则

1. **个性化**：至少包含候选人 1 个真实信息点（公司名/技术方向/经历）
2. **简洁直接**：100-180 字，严格小于 300 字
3. **平等语气**：不卑不亢，不过度吹捧，不群发口吻
4. **目的明确**：一句话说清楚找他/她的原因

---

## 消息结构模板

```
[个性化开场] + [岗位简介] + [为什么找你] + [行动引导]
```

### 示例结构（技术岗）

```
你好，看到你在 [公司名] 做 [技术方向] 的经历，
我们正在招 [岗位名]，[一句话描述岗位核心]。
你的 [某个具体背景] 和我们的方向很匹配，
有兴趣了解一下吗？
```

### 示例结构（产品/业务岗）

```
你好，看到你有 [行业/业务] 的经验，
我们在做 [业务方向]，正在招 [岗位名]。
[一句话说为什么找这个人]，
方便聊聊吗？
```

---

## 质量检查清单

发送前确认：

- [ ] 包含候选人真实信息（非模板填充）
- [ ] 字数在 100-300 字之间
- [ ] 没有"您好！我司……"式的群发开头
- [ ] 没有过度吹捧（"非常优秀"、"行业大牛"）
- [ ] 岗位名称正确
- [ ] 没有错别字

---

## 常见错误示例

❌ **群发口吻**
> 您好！我公司正在招聘优秀的算法工程师，您的背景非常契合，欢迎了解！

❌ **过度吹捧**
> 拜读了您的简历，您的履历令人印象深刻，绝对是行业顶尖人才……

❌ **信息量为零**
> 你好，我们有个职位你可能感兴趣，方便聊聊吗？

✅ **正确示例（替换为真实信息）**
> 你好，看到你在 [公司] 做 [具体技术方向]，
> 我们在 [业务场景] 上有类似的挑战，正在招 [岗位名]。
> 你在 [某个项目/经历] 上的经验和我们现阶段需求很贴合，
> 有兴趣了解一下吗？

---

## 跟进话术（已读未回）

**第一次跟进**（3 天后）：
```
你好，上次发了消息没收到回复，
不知道是不是没看到，
[岗位] 的机会还在，有兴趣的话随时聊～
```

**第二次跟进**（7 天后，若仍无回复则不再跟进）：
```
最后补充一下，[岗位] 目前在积极推进，
如果时机不合适也完全理解，
有需要随时找我～
```

---

## 拒绝候选人（礼貌感谢）

```
感谢你的时间，这次可能暂时不太匹配，
但你的 [某个背景] 印象很深，
有合适机会再联系你，祝一切顺利～
```
