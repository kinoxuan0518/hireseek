# Keyword Map

先按技术主题选词，再按渠道翻译成中英文变体。每次至少同时准备：

- 核心词
- 技术词
- 证据词
- 排除词

## 1. RL / Post-training

核心词：

- 强化学习
- RL
- RLHF
- 对齐
- 后训练

技术词：

- PPO
- DPO
- reward model
- preference model
- policy optimization
- SFT
- evaluation

证据词：

- reward
- preference data
- alignment
- judge model
- offline eval
- online eval

排除词：

- 推荐系统优化
- 广告投放
- 量化交易
