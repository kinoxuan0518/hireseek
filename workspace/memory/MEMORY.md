# HireSeek 自动记忆

这个文件会自动记录 HireSeek 学到的招聘经验、用户偏好、常见模式等。
每次启动时，这些内容会注入到系统提示中。

## 使用规则

- 记录稳定的模式和约定（跨多次对话验证）
- 记录关键决策、重要文件路径、项目结构
- 记录用户偏好（工作流、工具、沟通风格）
- 记录常见问题的解决方案
- **不要**记录会话临时信息
- **不要**记录未验证的推测
- **不要**记录与 SOUL.md/PLAYBOOK.md 重复的内容
