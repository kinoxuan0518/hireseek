# 收件箱扫描（Inbox Scan）

## 任务目标

扫描 BOSS直聘的消息收件箱，找出所有**候选人主动回复过消息**的对话，输出姓名列表。不发送任何消息，只读取和记录。

## 执行流程

1. 确认当前在 BOSS直聘消息页面（`/web/im/`）
2. 滚动浏览对话列表，逐一检查
3. 判断标准：对话中**最后一条消息是候选人发的**（不是我们发的），即视为已回复
4. 记录这些候选人的姓名
5. 滚动到底部，确保覆盖所有近期对话（近 30 天内）
6. 输出结果

## 输出格式（严格遵守）

任务完成后，按以下格式输出，每行一个姓名：

```
已回复候选人:
- 张三
- 李四
- 王五

扫描总数: <数字>
已回复数: <数字>
```

如果没有任何回复，输出：
```
已回复候选人:
（无）

扫描总数: <数字>
已回复数: 0
```

## 注意事项

- 只看，不发消息
- 系统消息、平台通知不算候选人回复
- 如果对话列表很长，优先处理近 14 天内的
